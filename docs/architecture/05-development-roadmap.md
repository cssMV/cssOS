# cssMV Minimum Viable Development Roadmap v1

## Guiding Rule

Build the skeleton first, then the MV path, then microdrama, then series and cinema.

## Phases

### Phase A: Shared Skeleton

Implement first:

- mode definitions
- capability definitions
- `ProjectSpec`
- `StoryGraph` MVP
- `OutputPackage`
- `CssMVEngine`
- directory skeleton

### Phase B: MV End-to-End Path

Implement next:

- `InputAdapter`
- lightweight `StoryEngine`
- `MVPlanner`
- `SceneComposer`
- `MusicDirector`
- `MediaCore`
- `OutputPackager`

### Phase C: Microdrama Integration

Add after the MV path stabilizes:

- `ShortDramaCompiler`
- `EpisodeFlow`
- basic `CharacterMemory`
- `MusicFragmentDirector`
- rules-based `HookEngine`
- `MusicContinuationCard`

### Phase D: Series and Cinema Expansion

Add after microdrama MVP:

- `SeriesPlanner`
- `CinemaPlanner`
- `BranchDirector`
- `EndingResolver`
- full `threads`, `branches`, and `endings` schema

## Milestones

1. Shared skeleton complete
2. MV MVP complete
3. microdrama MVP complete
4. series and cinema extension-ready
